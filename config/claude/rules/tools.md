# Preferred Tools

Use these tools instead of their standard alternatives:

| Tool   | Replaces | Description          |
| ------ | -------- | -------------------- |
| `zsh`  | bash     | Shell                |
| `rg`   | grep     | Fast search          |
| `fd`   | find     | File finder          |
| `bat`  | cat      | Syntax highlighting  |
| `eza`  | ls       | Git-aware listing    |
| `gomi` | rm       | Safe delete to trash |
| `jq`   | -        | JSON processor       |
| `gh`   | git      | GitHub CLI           |
