# Raycastの設定について

## Overview

- Raycast Configだと連携したくない情報までExport/Importしてしまうため、最低限の情報をDoc化

## Keymap

- leader: <Cmd + Opt>

### Applications

| application       | description          | keymap             |
| ----------------- | -------------------- | ------------------ |
| WezTerm           | primary Terminal     | <leader> + <Space> |
| Warp              | secondary Terminal   | <leader> + w       |
| Google Chrome     | browser              | <leader> + g       |
| Cursor            | IDE                  | <leader> + c       |
| VSCode            | IDE                  | <leader> + v       |
| DBeaver           | GUI DB Client        | <leader> + d       |
| Things            | ToDo Management      | <leader> + t       |
| Activity Monitor  |                      | <leader> + a       |
| Clipboard History | Clipboard Management | <Cmd> + <Cmd>      |

### Window Management

| actions      | keymap           |
| ------------ | ---------------- |
| Left Half    | <leader> + left  |
| Right Half   | <leader> + right |
| Maximize     | <leader> + down  |
| Next Display | <Shift> + w      |
